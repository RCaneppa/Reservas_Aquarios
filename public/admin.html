📸 Fotos dos espaços

Faça upload de uma foto real para cada espaço (JPG, PNG ou WEBP, até 3 MB). Espaços sem foto específica continuam usando a imagem genérica do tipo.

⚠️ Atenção: no plano gratuito do Render, fotos enviadas podem ser perdidas em redeploys. Para produção, configurar Render Disk persistente ou storage externo (S3/Cloudinary).
