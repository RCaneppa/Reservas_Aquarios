📥 Importação em massa (.xlsx)

Baixe o modelo, preencha com os dados dos sócios e faça o upload. Colunas: matricula, nome, cpf, email, telefone, senha, papel, adimplente. Senhas em branco serão geradas automaticamente no formato <matricula>@aqua.

⬇️ Baixar modelo .xlsx Nenhum arquivo selecionado

🔄 Atualização delta de adimplência (.xlsx)

Carga delta: atualiza somente os sócios listados na planilha. Quem não aparecer no arquivo permanece como está. Colunas obrigatórias: matricula e adimplente (1 = em dia, 0 = inadimplente). Aceitam-se também: "sim/não", "ok/pendente", "em dia/inadimplente".

⬇️ Baixar modelo Nenhum arquivo selecionado

📸 Fotos dos espaços

Faça upload de uma foto real para cada espaço (JPG, PNG ou WEBP, até 3 MB). Espaços sem foto específica continuam usando a imagem genérica do tipo.

⚠️ Atenção: no plano gratuito do Render, fotos enviadas podem ser perdidas em redeploys. Para produção, configurar Render Disk persistente ou storage externo (S3/Cloudinary).
